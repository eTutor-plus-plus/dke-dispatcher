Anleitung zum erstellen der Datenbanken für das SQL Modul auf PostgreSQL:

1. PostgreSQL installieren
2. Skript "databases DDl.sql" ausführen, um Datenbanken zu erstellen.
3. User "sql" mit passwort "sql" anlegen
4. Inhalt des Ordners /data in "C://Users/Public" kopieren
5. Skripts zum erstellen der Tabellen und laden der Daten ausführen
	. "sql DDL.sql" auf der sql-datenbank
	. "sql begin DDL.sql" sowohl auf sql_trial_begin als auch auf sql_submission_begin datenbank 
	.etc.
